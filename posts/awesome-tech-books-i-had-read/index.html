Awesome Programming books

This repo colllects some awesome 👍 books (i thought 🤔) for several topics like Machine Learning, Deep Learning.

Machine Learning

Book Title Recommended Comments
《Probabilistic Machine Learning: An Introduction》 by Kevin Patrick Murphy. (2021) ⭐⭐⭐⭐⭐ 神书MLAPP(2021)的作者在2021年又公开一本PML系列新书,涉及内容非常广泛并且内容与时俱进，以至于一本书写不下(2022年将出版PML系列的第三本书，敬请期待)

Deep Learning

Book Title Recommended Comments
《Deep Learning With Pytorch》 by Eli Stevens, Luca Antiga, Thomas Viehmann. (2020) ⭐⭐⭐⭐⭐ 官方出品，必属精品
《Dive into Deep Learning》by Aston Zhang, Zachary C. Lipton, Mu Li, and Alexander J. Smola ⭐⭐⭐⭐⭐ 被全球近200所大学作为课本或参考书，内容涵盖DL的多个领域，不仅有详细的数学公式，还有PyTorch,MxNet等多种框架的代码实现

Computer Vision

Book Title Recommended Comments
《深度学习之PyTorch物体检测实战》 by 董洪义 ⭐⭐⭐⭐⭐ 由浅入深，即使是初学者，也能迅速掌握动手实现基于深度学习的物体检测方法
